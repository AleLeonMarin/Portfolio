Maps

This project involves complex algorithms, data structures, Java development, CSS, and animations. It consists of a simulation of a Maps application. Nodes and edges are created, and the shortest path is calculated by Dijkstra's algorithm or the Floyd-Warshall algorithm. It also lets you close and open paths during execution, and it will automatically calculate the shortest path with the edges that are still open. More info in the GitHub repo.

A assemblyx86 16 bits program that simulates the game "Etch A Sketch". It uses the 8086 interrupts to draw on the screen. It also uses the keyboard to paint pixels in the grid previous marked. Finally it permits you to insert images that need to be converted into a 16-bmp image. The program contains a python to program to do it. More info in the GitHub repo.
